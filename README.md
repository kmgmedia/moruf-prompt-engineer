## How to work on this project locally

Requirements: Node.js (18+ recommended) and npm.

Quick start:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd moruf-prompt-engineer-portfolio-main

# Install dependencies
npm install

# Start the development server
npm run dev
```

You can also edit files locally in your preferred IDE and commit/push changes as usual.

- TypeScript

Build for production and deploy the `dist` output from `npm run build` using your preferred hosting provider.
